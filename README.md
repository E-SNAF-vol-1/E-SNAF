# 🛒 E-SNAF (E-Ticaret Platformu)

E-SNAF, modern web teknolojileriyle geliştirilmiş, kullanıcı dostu ve dinamik tema desteğine sahip bir e-ticaret platformudur. Proje, hem kullanıcılar hem de yöneticiler için optimize edilmiş bir alışveriş deneyimi sunar.

## ✨ Öne Çıkan Özellikler

* **🎨 Dinamik Tema Sistemi:** Kullanıcılar (veya admin) tarafından tek tıkla değiştirilebilen **Klasik Bej**, **Gece Modu (Dark)** ve **Okyanus** temaları.
* **🔍 Gelişmiş Ürün Detay:** Ürün resimlerine odaklanan, üzerine gelince büyüyen (zoom) ve yeni sekmede detaylı inceleme sunan modern arayüz.
* **🌤 Canlı Veri Entegrasyonu:**
    * Anlık döviz kurları (USD, EUR, ALTIN).
    * Kullanıcının konumuna göre anlık hava durumu bilgisi.
* **📱 Responsive Tasarım:** Tüm cihazlarda (Mobil, Tablet, Masaüstü) kusursuz görünüm.
* **🛒 Sepet ve Ödeme:** Hızlı ürün ekleme ve adım adım ödeme süreçleri.

## 🛠 Kullanılan Teknolojiler

* **Frontend:** React, Tailwind CSS v4, Vite, Axios.
* **Backend:** Node.js, Express.
* **Veritabanı:** Supabase / PostgreSQL.
* **Iframe Entegrasyonu:** Angular tabanlı mikro-servis ürün listeleme desteği.

## 🚀 Kurulum ve Çalıştırma

Projeyi kendi yerelinizde çalıştırmak için şu adımları izleyin:

1.  **Depoyu Klonlayın:**
    ```bash
    git clone [https://github.com/E-SNAF-vol-1/E-SNAF.git](https://github.com/E-SNAF-vol-1/E-SNAF.git)
    ```
2.  **Bağımlılıkları Yükleyin:**
    ```bash
    cd E-SNAF
    npm install
    ```
3.  **Uygulamayı Başlatın:**
    ```bash
    npm run dev
    ```
